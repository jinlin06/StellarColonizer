# Stellar Colonizer 星河殖民者

一款基于JavaFX的4X策略游戏，玩家将在广阔的宇宙中建立和发展自己的星际文明。

## 功能特点

- 六边形网格地图系统
- 殖民地建设和管理
- 资源管理系统
- 科技树研究
- 多个AI对手
- 回合制和实时游戏模式

## 技术栈

- Java 17
- JavaFX
- Maven

## 如何运行

1. 确保已安装 Java 17+ 和 Maven
2. 克隆项目到本地
3. 在项目根目录执行以下命令：

```bash
mvn clean javafx:run
```

## 游戏玩法

1. 建立殖民地并发展经济
2. 研究新技术提升实力
3. 与其他派系进行外交或战争
4. 扩张领土探索未知星系
5. 实现各种胜利条件

## 项目结构

- `core/` - 核心游戏逻辑
- `model/` - 游戏对象模型
- `view/` - UI界面组件
- `controller/` - 控制器逻辑
- `util/` - 工具类
